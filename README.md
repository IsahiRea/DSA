# DSA Portfolio

A React-based portfolio demonstrating Data Structures and Algorithms concepts through interactive examples and visualizations.

## Features

- **Data Structures**: Arrays, Linked Lists, Stacks, Queues, Trees, Graphs
- **Algorithms**: Sorting, Searching, Recursion, Dynamic Programming
- **Interactive Visualizations**: Step-by-step demonstrations of concepts
- **Responsive Design**: Mobile-first approach with desktop enhancements

## Tech Stack

- React 19
- React Router
- Vite
- CSS (custom design system)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── layouts/        # Page layouts (MainLayout)
├── pages/          # Route components (Home, Topics, About)
├── components/     # Reusable UI components
└── index.css       # Global styles and design tokens
```

## Design

The site uses a "Terminal Elegance" design theme featuring:
- Syne + JetBrains Mono typography
- Dark theme with cyan/purple/pink accents
- Subtle animations and glow effects
- Grid-based responsive layouts
